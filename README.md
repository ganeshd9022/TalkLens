# TalkLens 👁️🎙️

TalkLens is an AI-powered assistive system designed to help **blind, deaf, and mute individuals** interact with their environment and society using **Computer Vision, Speech, and AI**.

## Phase 1 – Vision Assistance
- Real-time object detection
- Scene understanding
- Voice-based querying
- Clear, stable audio feedback

## Tech Stack
Python | YOLOv8 | Speech Recognition | Text-to-Speech | Streamlit
